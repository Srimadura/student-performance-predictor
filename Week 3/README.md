# Week 3 - LSTM Time Series Prediction

## Objective
Predict the next week's student performance using an LSTM model based on historical attendance and study trends.

## Dataset
The dataset contains weekly records of:
- Week
- Semester
- Attendance
- Study Hours
- Performance Score

## Model
- LSTM (Long Short-Term Memory)

## Libraries Used
- TensorFlow
- Pandas
- NumPy
- Scikit-learn

## Output
The model predicts:
- Next Week Performance Score
- Performance Category (Good/Average/Poor)
